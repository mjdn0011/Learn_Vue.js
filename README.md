#### # Vuex
##### Vuex 라이브러리의 주요 속성 [S.G.M.A]
* **state**: 여러 컴포넌트에 공유되는 데이터(상태) - `data`
* **getters**: 연산된 state 값을 접근하는 속성 - `computed`
* **mutations**: state 값을 변경하는 이벤트 로직(메서드) - `method`
* **actions**: 비동기 처리 로직을 선언하는 메서드 - `async method`
